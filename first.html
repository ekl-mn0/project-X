HTML. Теги и их использование

Метаданные документа
<html></html> корневой элемент html-документа
<head></head> контейнер для метаданных html-документа
<title></title> заголовок/ имя html-документа
<base> базовый url-адрес, относительно которого вычисляются относительные адреса
<link> подключает внешние таблицы стилей
<meta> мета-данные веб-страницы
<style></style> корневой элемент html-документа
Группировка содержимого
<p></p> параграфы в тексте
<address></address> контактные данные автора документа или статьи
<hr></hr> горизонтальная линия
<pre></pre> выводит текст с пробелами и переносами
<blockquote></blockquote> большая цитата
<ol></ol> упорядоченный нумерованный список
<ul></ul> маркированный список
<li></li> элемент списка
<dl></dl> контейнер для термина и его описания
<dt></dt> задаёт термин
<dd></dd> расшифровывает термин
<figure></figure> независимый контейнер для такого контента как изображения, диаграммы и т.п.
<figcaption></figcaption> заголовок для элемента
<main></main> контейнер для уникального основного содержимого в пределах одной страницы сайта
<div></div> контейнер для разделов html-документа, группирует блочные элементы
Интерактивные элементы
<details></details> контейнер с дополнительными сведениями, который можно открыть или закрыть
<summary></summary> видимый заголовок для элемента <details>
<dialog></dialog> диалоговое окно, инспектор или окно
Встроенное содержимое
<picture></picture> контейнер для одного <img> и ноль или больше <source>
<source> местоположение и тип альтернативных медиаресурсов для элементов <picture>, <video>, <audio>
<img> html-изображения
<iframe></iframe> создаёт встроенный фрейм
<embed> встраивает внешний интерактивный контент или плагин
<object></object> контейнер для встраивания мультимедиа
<param> задаёт параметры для плагинов, встраиваемых с помощью элемента <object>
<audio></audio> добавляет аудио-файлы
<video></video> добавляет видео-файлы
<track> субтитры для элементов <audio> и <video>
<map></map> активные области на карте-изображении
<area> гиперссылка с текстом или активная область внутри карты-изображения
Семантика текста
<a></a> гиперссылка
<s></s> перечёркивает неактуальный текст
<time></time> дата / время документа или статьи
<kbd></kbd> текст, вводимый пользователем с клавиатуры
<sub></sub> подстрочное написание символов
<sup></sup> надстрочное написание символов
<i></i> выделяет текст курсивом без акцента
<b></b> задает полужирное начертание отрывка текста, без дополнительного акцента
<u></u> выделяет отрывок текста подчёркиванием, без дополнительного акцента
<span></span> контейнер для строчных элементов
<br> перенос текста на новую строку
Секционные элементы
<body></body> тело html-документа
<article></article> раздел контента, образующий независимую часть документа или сайта
<section></section> логическая область (раздел) страницы, обычно с заголовком
<nav></nav> раздел документа, содержащий навигационные ссылки по сайту
<aside></aside> контент страницы, имеющий косвенное отношение к основному контенту

Табличные данные
<table></table> html-таблица
<caption></caption> ?подпись к таблице?
<colgroup></colgroup> контейнер для одного или нескольких <col>
<col> выбирает для форматирования столбцы
<thead></thead> блок заголовков таблицы
<tbody></tbody> тело таблицы
<tfoot></tfoot> нижний колонтитул таблицы
<tr></tr> строка таблицы
<td></td> ячейка таблицы
<th></th> заголовок столбца таблицы
Скрипты
<script></script> подключает сценарии к странице
<noscript></noscript> секция, не поддерживающая скрипт
<template></template> фрагменты HTML-кода, которые могут быть клонированы и вставлены в документ скриптом
<canvas></canvas> холст-контейнер для динамического отображения изображений
Правки в тексте
<del></del> перечёркивает текст, помечая как удаленный
<ins></ins> подчёркивает изменения в тексте
Формы
<form></form> html-форма
<label></label> текстовая метка для элемента <input>
<input> многофункциональные поля формы
<button></button> интерактивная кнопка
<select></select> элемент управления с выбором значений из предложенных вариантов <option>
